# 🎯 AI 智能面试助手 (Offer-AI)

基于 **阿里云 FunASR 实时语音识别** + **智谱 GLM-4 大模型** 的智能面试辅助工具，支持专业术语热词优化、实时问题检测与智能解答，助力面试过程中快速响应专业问题。

## ✨ 核心功能

### 🎤 实时语音转文字
- 采用 **阿里云 FunASR 实时语音识别** 技术
- 支持动态热词表，AI/面试专业术语识别率提升 30%+
- 100ms 低延迟音频流处理
- 自动语义标点补充

### 🧠 智能问题检测
- **双模式检测**：显性（带问号）+ 隐性问题识别
- **上下文理解**：基于 LLM 的语义分析，准确区分面试问题/闲聊/陈述
- **防误触发**：严格校验机制，避免重复回答和无效问题
- **实时响应**：问题检测后立即启动解答线程

### 💡 多线程并发解答
- 集成 **智谱 GLM-4.7-flash** 大模型
- 支持最多 5 个问题并发处理
- 流式输出答案，无需等待完整生成
- 智能控制回答长度（max_tokens=2048）

### 🌐 本地 Web 前端实时展示
- **双栏布局**：左侧实时语音转写，右侧 AI 分析结论
- **SSE 流式推送**：答案逐字显示，带光标闪烁效果
- **可折叠思考过程**：点击展开查看 AI 推理细节
- **暗色主题**：保护视力，适合长时间面试

### 📝 完善的日志系统
- 识别文本持久化 (`LLM_voice_txt.txt`)
- LLM 解答结果存档 (`LLM_response.txt`)
- 时间戳精确到毫秒
- 自动清理历史日志，迎接新对话

### 🔧 优雅的资源管理
- 退出时自动清理云端热词表
- 音频资源完整释放
- 异常中断保护机制
- 最多等待 5 秒完成正在进行的解答

## 🚀 快速开始

### 环境准备
```
conda create -n offer-ai python=3.11
conda activate offer-ai
pip install -r requirements.txt
```

### 配置文件

在项目根目录创建 `.env` 文件：
```
DASHSCOPE_API_KEY=your_dashscope_api_key
ZHIPUAI_API_KEY=your_zhipuai_api_key
```
### 启动服务
```
python voice_q_a_funasr.py
```
启动后访问：**http://localhost:8848/monitor.html**
